# ⚡ NEON DRIFT

A premium synthwave endless arcade game. Dodge neon obstacles and survive as long as possible.

## Controls
- **Hold** screen / spacebar / right arrow → Ball drifts RIGHT
- **Release** → Ball drifts LEFT
- **P / Escape** → Pause

## Themes
- 🟣 Cyberpunk
- 🔵 Space
- 🌸 Vaporwave
- 🟢 Matrix
- 🔴 Blood Moon
